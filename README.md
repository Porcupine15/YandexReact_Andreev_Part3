**Проект**: Страница блога с настраиваемым оформлением через панель

## Используемые технологии

- HTML
- SCSS
- JavaScript + TypeScript

---
## Основные возможности

- **Гибкая кастомизация**:
  - **Выбор шрифта**
  - **Настройка размера шрифта**: через радиокнопки
  - **Изменение цвета текста и фона**: из готовых цветовых схем
  - **Регулировка ширины контента**

- **Интерактивная панель**:
  - Открытие и закрытие по **клику** на кнопку со стрелкой
  - Автоматическое закрытие при клике **вне панели**

- **Применение и сброс изменений**:
  - Настройки вступают в силу **после** нажатия кнопки «Применить»
  - Возврат к стандартным параметрам с помощью кнопки «Сбросить»

- **Сохранение состояния**:
  - При повторном открытии панели сохраняются последние выбранные настройки

- **CSS-переменные**:
  - Используются для удобного управления стилями и кастомизации

---
## Особенности реализации

- Сайдбар открывается по клику на «стрелку», закрывается повторным нажатием или кликом на оверлей
- Изменения вносятся только после нажатия кнопки «Применить», без мгновенного эффекта
- Сброс настроек возвращает параметры по умолчанию
- Стили реализованы на базе CSS-переменных
- Последние выбранные параметры сохраняются между открытиями панели

## Компонент ArticleParamsForm

Форма для настройки параметров статьи, включая:
- **Шрифт** 
- **Размер шрифта**
- **Цвет шрифта** 
- **Цвет фона** 
- **Ширина контента**

### Функциональность

1. **Открытие и закрытие панели**: Управляется состоянием open, также срабатывает при клике вне панели.
  
2. **Применение изменений**: По нажатию кнопки «Применить» изменения сохраняются в родительском состоянии

3. **Сброс настроек**: Кнопка «Сбросить» возвращает все параметры к значениям по умолчанию

4. **Используемые хуки**:
   - `useState` - Управление состоянием
   - `useRef` - Отслеживание кликов вне панели

### Как использовать

1. Импортируйте компонент в проект.
2. Передайте необходимые props:
   - `articleState`: объект с текущими настройками
   - `setArticleState`: функция для обновления состояния
